For builders

Protect any AI or data API with micropayments.

The reusable piece is the Paid Intelligence API Kit: pricing tiers, query fingerprints, invoice binding, access tokens, wallet entitlements, and Arcscan settlement tracking. QMA is the first example provider, not the whole platform.

Open source

The public repo includes the frontend, FastAPI provider API, Paid Intelligence Kit, sample datasets, and crawler scripts. The full production dataset stays private and loads through environment paths on the deployed backend.
